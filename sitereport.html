Site Report

This report provides an overview of my experience developing this portfolio website, including the learning process, development stages, design decisions, and validation results.

Learning Experience

Throughout this term, I have learned the basics of web development, including HTML for structuring content, CSS for styling, and ensuring responsive design. One of the biggest challenges was mastering CSS Grid and Flexbox for layout purposes. However, the process of overcoming these challenges greatly enhanced my understanding and skills.

Development Process

The development process began with planning the structure and content of the website. I created wireframes to visualize the layout and used them as a guide while coding. Regular testing and debugging were crucial to ensure each page functioned correctly and met the assignment requirements.

Design Decisions

I chose a clean, modern design with a consistent color scheme of blue and white, which conveys a professional look. The font selection aimed to provide readability and aesthetic appeal. I drew inspiration from several design blogs and websites, such as Smashing Magazine and CSS-Tricks. The responsive design was implemented using media queries to ensure the site works well on both desktop and mobile devices.
